# 配置github.com
Host github.com                 
    HostName github.com
    IdentityFile C:\\Users\\user\\.ssh\\id_rsa_yx
    PreferredAuthentications publickey
    User yx

    IdentityFile C:\\Users\\user\\.ssh\\id_rsa
